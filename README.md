## 📌 Projects

### 🌾 [AgroVision — Unkrauterkennung & VRA‑Kartengenerierung](https://github.com/jeorgesilva/AgroVision)
KI‑System zur automatisierten Objekterkennung für die Generierung von Entscheidungsdaten (VRA‑Karten). Bewertung und Optimierung der Schlussfolgerungsgenauigkeit unter variierenden Feld‑ und Aufnahmebedingungen.  
**Tech Stack:** Python, Ultralytics YOLO, OpenCV, GeoPandas, Streamlit.

### 📚 [RAG‑gestützter KI‑Tutor (Teacher_Isa)](https://github.com/jeorgesilva/Teacher_Isa)
End‑to‑end NLP‑Pipeline auf Basis der RAG‑Architektur zur Strukturierung von Bildungsdaten und zur Erhöhung der Genauigkeit von LLM‑Antworten. Skalierbare RESTful‑APIs für nahtlose Integration und semantischen Abruf.  
**Tech Stack:** Python, LangChain, FastAPI, ChromaDB, Docker.

### 📖 [Lianes Library — Multiagenten Bibliothekssystem](https://github.com/jeorgesilva/lianes-library)
Kreditmanagementsystem, das relationale SQL‑Datenbanken mit KI‑gestützter Abfrage verbindet. Implementierung semantischer Suche und stimmungsbasierter Empfehlungen zur Verbesserung der Buchfindung und Nutzererfahrung.  
**Tech Stack:** Python, SQL, NLP, Semantische Suche, Sentiment‑Analyse.
